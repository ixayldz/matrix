# Matrix CLI v0.1.0

Release date: 2026-02-16

## Highlights
- Improved TUI stability and panel rendering behavior.
- Added provider-first auth onboarding flow in TUI and CLI.
- Added compatibility-focused command improvements (`/login`, `/auth use`, `/auth login`).

## TUI Improvements
- Fixed hook-order crash in `DiffViewer` that could force fallback mode.
- Reduced scroll jitter and duplicate rendering artifacts in panel layout.
- Improved command result visibility and provider/model status display.

## Auth and Onboarding
- Added cross-platform browser opener for provider login/key pages.
- `/new` now checks provider auth and guides key setup when missing.
- Added `/login [provider]` (`/link` alias) in TUI.
- Added CLI command: `matrix auth login [provider] --open`.

## Documentation
- Updated `README.md` onboarding and auth flow.
- Updated onboarding docs under `docs/onboarding/`.
- Added project changelog in `CHANGELOG.md`.
